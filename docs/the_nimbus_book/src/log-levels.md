# Change log level


You can customise Nimbus' verbosity with the `--log-level` option.

For example:

```
./run-mainnet-beacon-node.sh --log-level=WARN
```

The default value is `INFO`.

Possible values (in order of decreasing verbosity) are:

```
TRACE
DEBUG
INFO
NOTICE
WARN
ERROR
FATAL
NONE
```
